# LLM 提供商指南

LibreFang 包含全面的模型目录，覆盖 **3 个原生 LLM 驱动**、**20+ 提供商**、**51 个内置模型**。

---

## 快速设置

```bash
# 选择一个提供商 - 设置环境变量即可
export GEMINI_API_KEY="your-key"      # 免费额度可用
export GROQ_API_KEY="your-key"        # 免费额度可用
export ANTHROPIC_API_KEY="your-key"
export OPENAI_API_KEY="your-key"
```

LibreFang 在启动时自动检测哪些提供商已配置 API 密钥。

<Note>
**推荐**: Groq 和 Gemini 提供免费额度，适合测试和开发。
</Note>

## 每个 Agent 的模型覆盖

可以在 agent 级别覆盖默认模型：

```toml
[model]
provider = "groq"
model = "llama-3.3-70b-versatile"
```

---

## 模型路由

`ModelRouter` 根据任务复杂度自动选择模型：

- **简单任务** → 快速/便宜模型
- **复杂任务** → 智能/昂贵模型

---

## 成本追踪

LibreFang 跟踪每个模型的使用成本：

```bash
# 查看全局预算
curl http://127.0.0.1:4200/api/budget

# 查看每个 agent 的成本
curl http://127.0.0.1:4200/api/budget/agents
```

---

## 后备提供商

配置后备提供商以提高可靠性：

```toml
[[fallback_providers]]
provider = "groq"
model = "llama-3.1-70b-versatile"
api_key_env = "GROQ_API_KEY"
```
