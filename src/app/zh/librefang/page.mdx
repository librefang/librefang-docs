# LibreFang - Agent Operating System

LibreFang 是一个用 Rust 编写的开源 Agent 操作系统，是 OpenFang 的社区维护分支。

<Note>
- **官网**: [https://librefang.ai/](https://librefang.ai/)
- **GitHub**: [https://github.com/librefang/librefang](https://github.com/librefang/librefang)
- **特性**: 14 个 Rust crates，137K+ 代码行，16 层安全防护，7 个内置 Hands
</Note>

## 核心概念

### Hands - 自主工作的 Agent

Hands 是 LibreFang 的核心创新——预构建的自主能力包，独立运行、按计划执行，无需人工干预。

| Hand | 功能 |
|------|------|
| **Clip** | YouTube 视频下载、剪辑、自动生成字幕和垂直短视频 |
| **Lead** | 每日潜在客户发现、丰富信息、评分排序 |
| **Collector** | OSINT 级情报收集、目标监控、知识图谱构建 |
| **Predictor** | 超级预测引擎、置信区间、校准推理链 |
| **Researcher** | 深度自主研究、多源交叉引用、CRAAP 评估 |
| **Twitter** | 自主 Twitter/X 账号管理、内容创作、排程发布 |
| **Browser** | Web 自动化、填写表单、点击按钮、多步骤工作流 |

### 架构

```text
librefang-kernel      编排、工作流、计量、RBAC、调度

librefang-api         140+ REST/WS/SSE 端点
librefang-channels    40 个消息适配器

librefang-hands       7 个自主 Hands

librefang-desktop     Tauri 2.0 原生应用
```

## 安装

### Cargo 安装（推荐）

```bash
cargo install --git https://github.com/librefang/librefang librefang-cli
```

### 从源码构建

```bash
git clone https://github.com/librefang/librefang.git
cd librefang
cargo install --path crates/librefang-cli
```

<Note>
**提示**: 首次运行 `librefang init` 会自动创建配置文件并引导你完成初始设置。
</Note>

### Docker

```bash
docker pull ghcr.io/librefang/librefang:latest

docker run -d \
  --name librefang \
  -p 4200:4200 \
  -e ANTHROPIC_API_KEY=$ANTHROPIC_API_KEY \
  -v librefang-data:/data \
  ghcr.io/librefang/librefang:latest
```

```bash
librefang --version
```

## 快速开始

### 1. 初始化

```bash
librefang init
```

这会创建 `~/.librefang/` 目录和默认配置文件。

### 2. 配置 API Key

LibreFang 需要至少一个 LLM 提供商 API Key：

```bash
# Anthropic (Claude)
export ANTHROPIC_API_KEY=sk-ant-...

# 或 OpenAI
export OPENAI_API_KEY=sk-...

# 或 Groq (有免费额度)
export GROQ_API_KEY=gsk_...
```

### 3. 编辑配置

编辑 `~/.librefang/config.toml`：

```toml
[default_model]
provider = "groq"
model = "llama-3.3-70b-versatile"
api_key_env = "GROQ_API_KEY"

[network]
listen_addr = "127.0.0.1:4200"
```

### 4. 启动守护进程

```bash
librefang start
```

启动后访问 http://127.0.0.1:4200/ 查看仪表板。

### 5. 激活 Hand

```bash
# 激活 Researcher Hand
librefang hand activate researcher

# 查看状态
librefang hand status researcher
```

### 创建自定义 Agent

创建 `my-agent.toml`：

```toml
name = "my-assistant"
version = "0.1.0"
description = "A helpful assistant"
author = "you"
module = "builtin:chat"

[model]
provider = "groq"
model = "llama-3.3-70b-versatile"

[capabilities]
tools = ["file_read", "file_list", "web_fetch"]
memory_read = ["*"]
memory_write = ["self.*"]
```

然后生成：

```bash
librefang agent spawn my-agent.toml
```

### 内置工具

- `file_read` - 读取文件
- `file_list` - 列出目录
- `web_fetch` - 获取网页
- `web_search` - 搜索网页
- `memory_store` - 存储记忆
- `memory_recall` - 回忆记忆
- `shell` - 执行 shell 命令
- `browser` - 浏览器自动化

## CLI 命令参考

### Agent 管理

```bash
# 生成 Agent
librefang agent spawn agents/hello-world/agent.toml

# 列出所有 Agent
librefang agent list

# 与 Agent 聊天
librefang agent chat <agent-id>

# 终止 Agent
librefang agent kill <agent-id>
```

### Hand 管理

```bash
# 列出所有 Hands
librefang hand list

# 激活 Hand
librefang hand activate researcher

# 查看状态
librefang hand status researcher

# 暂停 Hand
librefang hand pause researcher
```

```bash

# 创建工作流
```

## 16 层安全系统

LibreFang 采用深度防御策略，包含 16 个独立的安全层：

1. **WASM 双计量沙箱** - 燃料计量 + 时代中断
2. **Merkle 哈希链审计追踪** - 加密链接的完整操作链
3. **信息流污点追踪** - 敏感信息从源到汇的追踪
4. **Ed25519 签名 Agent 清单** - 加密签名的身份和能力集
5. **SSRF 保护** - 阻止私有 IP 和云元数据端点
6. **Secret 零化** - 自动清除内存中的 API 密钥
7. **OFP 双向认证** - HMAC-SHA256 验证
8. **能力门** - 基于角色的访问控制
9. **安全头** - CSP、X-Frame-Options、HSTS
10. **健康端点重写** - 公开/私有健康检查分离
11. **子进程沙箱** - 环境清理 + 进程树隔离
12. **提示注入扫描器** - 检测覆盖尝试
13. **循环守卫** - SHA256 循环检测
14. **会话修复** - 7 阶段消息历史验证
15. **路径遍历防护** - 规范化和符号链接转义防止
16. **GCRA 速率限制器** - 成本感知的令牌桶
