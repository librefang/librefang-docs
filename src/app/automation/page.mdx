# Team Yldm 自动化生命周期

系统不是冷冰冰的定时器而是根据你的生活节奏和基建压力自动调节的有机体

---

##  07:00 - 09:00觉醒与情报汇总
系统通过 `daily-morning-precache` 提前收集数据为你开启一天
- **07:30 每日简报**天气日程银行余额快递状态推送至 TG
- **08:00 技术晨报**GitHub Trending + RSS 高质量摘要
- **08:30 DevOps 晨报**K8s 集群健康度 + 核心站点可用性报告

##  09:00 - 18:00极速生产模式
在此期间系统的高频监控全面开启
- **PCU 自动化**每隔数小时扫描社区动态自动执行 `pcu-daily-scan`
- **实时审计**Lyra 对正在进行的代码 PR 执行 7 维质量审查（类型安全/错误处理/安全/性能/a11y/规范/测试）
- **健康巡检**每 30 分钟执行一次 `infra-service-monitor`

##  18:00 - 23:00复盘与生活提醒
辅助你平稳结束一天
- **18:00 视频推荐**YouTube 编程/科技视频定向分发
- **22:00 垃圾分类**根据横滨市日历推送分类提醒
- **23:30 晚间预览**生成明天的任务预告

##  00:00 - 06:00自我维护与进化
在你入睡后智多星接管系统进行“深度大修”
- **01:00 AI SRE 诊断**全量扫描 K8s 日志分析潜在性能瓶颈
- **02:00 安全审计**运行 `security_audit.sh` 确保无 API 密钥泄露
- **03:00 全量备份**将 OpenClaw 状态打包备份至 NAS 安全卷
- **04:00 技能进化**智多星消化当天的错误日志，自动产出优化补丁并更新长期记忆

---
*你可以随时发送指令 `@智多星 给我看任务清单` 来获取当前的 Cron 状态*
